# Copilot Money Clone (React Native + Expo)

A high-quality mobile app inspired by Copilot Money, built with React Native, Expo, and a local-first architecture.

This project focuses on:

- clean UX
- fast performance
- offline-first data
- strong architecture

---

## ✨ Features

- Transaction tracking and categorization
- Accounts and net worth overview
- Recurring expenses
- Savings goals
- Budgets and spending insights
- Cash flow visualization
- Investment tracking

The goal is to replicate the experience of a modern personal finance app where everything is **fast, visual, and easy to understand**.

---

## 🧱 Tech Stack

- React Native + Expo
- TypeScript
- pnpm
- SQLite (expo-sqlite)
- Drizzle ORM
- React Query (for DB queries)
- @shopify/flash-list (performance lists)
- victory-native-xl (charts)
- Unistyles (theming)

---

## 📦 Installation

```bash
pnpm install
```

---

## ▶️ Running the App

```bash
pnpm expo start
```

Then:

- Press `i` for iOS simulator
- Press `a` for Android
- Or scan QR with Expo Go

---

## 🗂 Project Structure

The app uses a **feature-based architecture** with:

- isolated features
- shared UI components
- local database layer

---

## 🧠 Architecture Highlights

- Local-first database using SQLite + Drizzle
- React Query for efficient data access
- Feature modules with clear boundaries
- Service layer to isolate business logic

## 🎨 Design Goals

- Clean financial data visualization
